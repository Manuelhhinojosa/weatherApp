# Weather app

This weather application enables users to enter the name of a city and retrieve current weather information. Built with HTML, CSS, and JavaScript, the app uses Axios to make requests to a weather API. Upon entering a city, the app fetches and displays relevant weather data, including temperature, humidity, and conditions. The user-friendly interface provides real-time weather updates, making it easy to check conditions for any city.

## Technologies used

- HTML
- CSS
- JavaScript
- Axios
- Weather API
- DOM manipulation

The app can be accessed here: https://weathermhh.netlify.app/
